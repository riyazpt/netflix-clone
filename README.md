# Netflix-simulation

Small Project Like Netflix

## Description

This project uses tmdb API(The movie database) and shows netflix like interface where movie trailer
would be shown in youtube link when clicked.

## Installing

Clone GitHub repo for this project locally
change directory to project folder

### `npm install`
